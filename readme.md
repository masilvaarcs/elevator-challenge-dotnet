# Elevator Challenge - .NET

Desafio técnico implementado em **C# WinForms** (.NET 4.7.2): sistema de análise estatística de elevadores do prédio 99a da Tecnopuc.

## Desafio

Dado um levantamento de uso de 5 elevadores (A-E) em 16 andares (0-15) com 3 turnos (M/V/N), extrair:

- Andar menos utilizado
- Elevador mais/menos frequentado e respectivo período
- Período de maior utilização do conjunto
- Percentual de uso por elevador

## Stack

- C# / .NET Framework 4.7.2
- WinForms
- Newtonsoft.Json
- LINQ

## Como executar

Abra `DesafioApisul.sln` no Visual Studio e execute (F5).
